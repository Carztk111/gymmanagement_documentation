ADMINISTRATION

Overview

GNEX 360 provides administrators with complete control over gym operations through a centralized management system. Administrators are responsible for processing membership applications, managing members and employees, monitoring attendance records, maintaining membership plans, generating reports, and overseeing daily gym activities.

MEMBER APPROVALS

Waitlist Management

Membership applications submitted by prospective members are placed in the waitlist module for administrator review. Administrators may approve, reject, or continue reviewing applications depending on the information provided.

Reviewing Applications

  1. Open the Waitlist module.
  2. Select an applicant from the pending list.
  3. Review personal information and registration details.
  4. Check the selected membership plan.
  5. Verify that the submitted information is complete.
  6. Continue reviewing other applicants if necessary.

Verifying Payment Information

  1. Review the applicant's payment information.
  2. Confirm that the payment transaction is valid.
  3. Verify the selected membership plan.
  4. If discrepancies are found, continue reviewing the application.
  5. Proceed to approval or rejection when verification is complete.

Processing Applications

  1. Review the completed application.
  2. Determine the appropriate action.
  3. If the applicant satisfies the requirements, select Approve.
  4. If the application contains invalid information or payment issues, select Reject.
  5. If additional review is required, postpone the decision and proceed to other applications.
  6. Confirm the selected action.

Activating Member Accounts

  1. Review approved applications.
  2. Verify that the membership plan is assigned correctly.
  3. Allow the system to generate the member account.
  4. Confirm that the member appears in the Members module.
  5. Continue processing other approved applications if necessary.
MEMBER RECORDS

Member Management

The Members module allows administrators to review member information, monitor member activities, and maintain account status throughout the member lifecycle.

Viewing and Updating Member Information

  1. Open the Members module.
  2. Browse or search for a member account.
  3. Select the desired member.
  4. Review personal information and membership details.
  5. Click Edit Member.
  6. Modify the necessary information.
  7. Save the changes.
  8. Verify that the updated information appears correctly.

Reviewing Member Records

  1. Navigate to the Members module.
  2. Select a member account.
  3. Review attendance logs.
  4. Review transaction history.
  5. Review recorded violations, if applicable.
  6. Verify that all records are accurate.

Managing Member Status

  1. Open the Members module.
  2. Select the member account.
  3. Review membership information and activity records.
  4. Determine whether the account should remain active.
  5. Click Deactivate Member when necessary.
  6. Confirm the action.
  7. Verify that the member status has been updated.

SUBSCRIPTION MANAGEMENT

Membership Management

The Membership module enables administrators to maintain membership plans, process renewal requests, and monitor subscription status.

Reviewing Membership Plans

  1. Open the Plans module.
  2. Review the list of available membership plans.
  3. Select a plan to view its details.
  4. Check the plan name, duration, and pricing information.
  5. If changes are required, select Edit Plan.
  6. If the plan is no longer needed, select Delete Plan.
  7. If a new plan is required, select Create Plan.
  8. Save any modifications made.
  9. Verify that the plans list reflects the updated information.

Creating Membership Plans

  1. Open the Plans module.
  2. Select Create Plan.
  3. Enter the plan name.
  4. Specify the duration and pricing.
  5. Configure additional plan details.
  6. Review the entered information.
  7. If changes are required, modify the plan information before saving.
  8. If the plan should not be created, cancel the operation.
  9. Select Save to create the plan.
  10. Verify that the new plan appears in the plans list.
  11. If additional plans are needed, repeat the process.

Updating Membership Plans

  1. Open the Plans module.
  2. Select the membership plan to be modified.
  3. Review the current plan information.
  4. Update the pricing, duration, or other plan details as necessary.
  5. Review the modified information for accuracy.
  6. Make additional adjustments if further changes are required.
  7. Select Save to apply the changes, or cancel the operation if no updates are needed.
  8. Verify that the updated information is reflected in the plans list.
  9. Continue updating other plans if necessary.

Removing Membership Plans

  1. Open the Plans module.
  2. Select the desired plan.
  3. Review whether the plan is no longer needed.
  4. Choose Delete Plan.
  5. Confirm the action.
  6. Verify that the plan has been removed.

Processing Renewal Requests

  1. Open the Plans module.
  2. Review pending renewal requests.
  3. Verify payment information.
  4. Confirm the selected membership plan.
  5. Approve the renewal request.
  6. Verify that the member subscription has been updated.

Monitoring Expiration Dates

  1. Navigate to the Plans module.
  2. Review active subscriptions.
  3. Identify memberships nearing expiration.
  4. Review renewal requests if available.
  5. Monitor inactive or expired memberships.
  6. Take appropriate action when necessary.

Attendance Monitoring

Viewing Daily Attendance

  2. Select Daily Attendance.
  3. Choose the desired date.
  4. Review all recorded member visits.
  5. Verify successful check-ins.
  6. Monitor gym activity throughout the day.

Reviewing Attendance Logs

  1. Navigate to the Attendance module.
  2. Select Attendance Logs.
  3. Specify a date range.
  4. Review historical attendance records.
  5. Locate a specific member if necessary.
  6. Verify attendance information.

Identifying Peak Hours

  1. Open the Attendance module.
  2. Select Attendance Statistics.
  3. Review hourly attendance data.
  4. Identify periods with the highest activity.
  5. Analyze facility usage patterns.
  6. Use the information for scheduling and operations.

Analyzing Attendance Trends

  1. Navigate to Attendance Statistics.
  2. Select the preferred time period.
  3. Review attendance charts and summaries.
  4. Compare current and previous records.
  5. Identify increases or decreases in attendance.
  6. Use the information for planning purposes.

Investigating Attendance Issues

  1. Open Attendance Logs.
  2. Locate the affected member record.
  3. Review the recorded check-in information.
  4. Compare attendance details with membership status.
  5. Determine the cause of the discrepancy.
  6. Apply the necessary correction if required.
QR VALIDATION

QR Check-In Administration

The QR Administration module enables administrators to monitor attendance validation, verify membership eligibility, and resolve QR-related issues.

Verifying Member Entry

  1. Open the QR Scanner module.
  2. Scan the member's QR code.
  3. Review the member information displayed.
  4. Verify whether the membership is active.
  5. If the membership is valid, allow entry.
  6. The system automatically records attendance.

Reviewing Attendance Logs

  1. Navigate to the Attendance module.
  2. Select Attendance Logs.
  3. Choose the desired date range.
  4. Review member check-in records.
  5. Verify attendance history when necessary.

Resolving Failed Scans

  1. Review the failed scan notification.
  2. Identify the affected member account.
  3. Check whether the membership has expired.
  4. Verify whether the QR code is valid.
  5. If necessary, update the member account.
  6. Retry the QR validation process.

Handling Expired Memberships

  1. Scan the member QR code.
  2. If the membership has expired, deny entry.
  3. Inform the member about the expiration.
  4. Direct the member to submit a renewal request.
  5. Approve the renewal when payment has been verified.
  6. Allow entry once the membership becomes active.

Investigating Attendance Discrepancies

  1. Open the Attendance module.
  2. Review attendance records.
  3. Locate the missing or incorrect entry.
  4. Compare logs with member information.
  5. Determine the source of the discrepancy.
  6. Apply the necessary corrections.
